  Usage Guidelines

  1. Primary actions: Use indigo→purple gradient
  2. Data visualizations: Use the 4 category colors (blue, purple, orange, emerald)
  3. Text hierarchy: slate-800 → slate-600 → slate-500
  4. Backgrounds: Keep light (white, slate-50) with subtle gradients
  5. Hover states: Add shadow and slight lift (hover:-translate-y-1)
  6. Borders: Use matching -100 shade (e.g., border-blue-100 for blue cards)
